# CLAUDE.md — Skills Hub

Community skills registry. Skills are prompt-layer playbooks injected into agents at build time — not tool functions, pure knowledge/instructions.

## Structure

```
skills/
  code-review/SKILL.md
  git-workflow/SKILL.md
  error-handling/SKILL.md
  ...
_template/SKILL.md          # Template for new skills
registry.json               # Auto-generated (SHA256 hashes, metadata)
```

## Skill Format

Each skill is a single `SKILL.md` with YAML frontmatter + Markdown body:

```markdown
---
name: code-review
version: 1.0.0
description: ...
tags: [review, quality]
---

# Instructions for the agent...
```

## Workflow

1. Create `skills/my-skill/SKILL.md` using `_template/SKILL.md`
2. Open PR → auto-checks (schema, tokens < 4000, injection scan, semver)
3. On merge → GitHub Action generates/updates `registry.json`
4. Users reference via `hub://my-skill@1.0.0` in agentfile.yaml

## No Build Process

Pure Git + Markdown. No npm/pip. The registry.json is generated by CI.
